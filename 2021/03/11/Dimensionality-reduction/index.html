Dimensionality reduction

Posted by Changtao Li on 2021-03-11

Why Dimensionality Reduction?

In machine learning, we usually write all our data into a matrix: design matrix:

N is the number of samples while D is the number of features of each sample. In general, we wish N to be larger than D. However, in some circumstances, we do encounter design matrixes which D is larger. Under such situations, some machine learning algorithms will not work, while some will cost much more time. To address this problem, some researchers put forward some methods to reduce the dimension of the data. This is why we need dimensionality reduction.

Raising the Dimension of Your Data

However, before talking about dimensionality reduction, it’s more necessary, in my opinion, to talk about raising the dimension of your data. Raising the dimension is commonly seen in linear regression and kernel methods. So, let’s talk about them.

Dimensionality Reduction?

As you can see, we often raise the dimension of our data. And we often need to reduce the dimension of our data. What are we doing here?

I would give my opinion here to help you get the fist of it. Or you’ll be even more perplexed when seeing an algorithm called kernel PCA.

We can see them, in my opinion, as one same thing. They just want to get a even more proper representation of our data. So there is no significant distinction between them whatsoever. This is the most important topic in more machine learning and you can see this from the name of ICLR(International Conference of Learning Representation).

There are so many different methods. As usual, I will talk some easiest important.

PCA

This is a projection method. But before we can project the training set onto lower-dimensional hyperplane, we first need to choose the right hyperplane. In fact, we always choose the axis that preserves the maximum amount of variance and it will most likely lose less information than the other projections. This is the simple idea behind it. However, the devil is in the detail. I suggest you to look up Mr. Murphy’s Book for more about it.

To do PCA, you can use either SVD from Numpy or PCA from Sci-kit Learn.

Kernel PCA

By using kernel PCA, we can perform complex nonlinear projections for dimensionality reduction.

LLE

Locally linear embedding is a powerful nonlinear dimensionality reduction method.

Conclusion

Dimensionality Reduction belongs to the preprocessing. This is not the end of you machine learning project. On the contrary, it’s just the beginning. After this step, machine learning algorithms will become less time consuming. However, their performance will be compromised. There is trade-off in this step and you should always consider this trade-off.
