Changtao's weekly log (12/25)

Posted by Changtao Li on 2020-12-23

My work of this week can be categorized as follows:

Installing CUDA and CUDNN

As we all know, CPU is a very powerful device and it’s the core of modern computers. Although CPU can do many complex tasks, it is not designed for repeated simple computations, which, indeed, are the essences of Machine Learning. Luckily, we have another powerful chip, aka GPU, to address this. However, to use GPU to accelerate the computation of a neural network, we must install some interpreter and libraries: CUDA and CUDNN.

I have always used Google Colab to train my network. Admittedly, Google Colab is an amazing product, it still has some drawbacks:

  1. The connection between my computer and the Google server is not stable. As we know, Google, actually, is banned in China.
  2. No free lunch theorem. I cannot use a Google Server with a GPU for more than 12 hours.
  3. It’s hard to tackle local data on the Google server.

Finally, I decided to install CUDA and CUDNN on my own laptop. (Though there are computers with pre-installed CUDA and CUDNN, I still want to do it all by myself: you cannot always rely on somebody else) The installing procedures go as follows:

  1. Check the driver version of the GPU. Old drivers cannot support latest CUDA.
  2. Install CUDA and never don’t install all the components of it, especially Visual Studio Integration.
  3. Extract CUDNN and paste them to the place where you installed CUDA.

Exploring the latent space of the generator

A well-trained generator of a GAN can generate the data we want, such as pictures and sound, by mapping the noise sampled from latent space. Generally speaking, the dimension of the latent space(we usually set as 100) is much lower than the dimension of the data of interest(take RGB photos for example, every pixel is an independent dimension). As a consequence, we can think GAN as a data-compress algorithm. Every element of the vectors in the latent space controls some abstract features. Again, take photos for example, we can generate a happy man face by manipulating the latent vectors of happy woman face, neutral man face and neutral woman face.
